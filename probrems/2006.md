# 2006年 早稲田大学 商学部 数学

## 大問1

ア〜ケにはいるべき数を，マーク解答用紙の該当する数字の部分に1つだけマークせよ。ただし，分数はすべて既約分数で答えよ。

1. 整式 \(ax^3+bx^2-2\) が，整式 \((x+1)^2\) で割り切れるとき，定数
   \[
   a=\boxed{\text{ア}},\quad b=\boxed{\text{イ}}
   \]
   である。

2. 座標空間の点 \(A(1,0,1),\ B(1,0,0),\ C(-1,0,\sqrt{3}),\ D(-1,0,0)\) および \(P(x,y,0)\) に対し，
   \[
   \angle APB=\angle CPD
   \]
   が成り立っている。このとき，\(x,y\) は次の式を満たす。
   \[
   (x-\boxed{\text{サ}})^2+y^2=\boxed{\text{エ}}
   \]

3. 正の整数 \(n\) に対し
   \[
   f(n)=\frac{4n+\sqrt{4n^2-1}}{\sqrt{2n+1}+\sqrt{2n-1}}
   \]
   と定義するとき，
   \[
   \sum_{n=1}^{60}f(n)=f(1)+f(2)+f(3)+\cdots+f(60)=\boxed{\text{オカキ}}
   \]
   である。

4. 
   \[
   \cos 40^\circ+\cos 80^\circ+\cos 120^\circ+\cos 160^\circ
   =-\frac{\boxed{\text{ク}}}{\boxed{\text{ケ}}}
   \]

## 大問2

\(xy\) 平面の放物線 \(C:y=x^2\) 上に異なる2点 \(A(a,a^2),\ B(b,b^2)\) をとる。ただし，\(a<b\) とする。放物線 \(C\) の点 \(A,B\) における接線の交点を \(P\) とする。このとき，次の問いに答えよ。

1. 点 \(P\) の座標を求めよ。
2. 放物線 \(C\) と点 \(A,B\) における接線で囲まれる領域の面積を \(S\)，放物線 \(C\) と直線 \(AB\) で囲まれる領域の面積を \(T\) とする。このとき \(\dfrac{S}{T}\) の値を求めよ。
3. \(S=\dfrac94\) となるように点 \(A,B\) が放物線 \(C\) 上を動くとき，点 \(P\) の軌跡を求めよ。

## 大問3

正の整数 \(n\) に対して，整数 \(f(x)\) が次の条件 (i), (ii) を満たすように定義されている。

\[
\text{(i)}\ f(1)=1
\]
\[
\text{(ii)}\ \text{任意の正の整数 }n\text{ に対して, }
\begin{cases}
f(2n)=f(n)\\
f(2n+1)=f(n)+1
\end{cases}
\]

このとき，次の問いに答えよ。

1. \(f(4),\ f(13),\ f(2^{10}+1)\) を求めよ。
2. \(1\le n\le 2006\) のとき，\(f(n)\) の最大値 \(M\) を求めよ。
3. (2)で求めた最大値 \(M\) に対して，\(f(n)=M\) を満たす \(2006\) 以下の正の整数 \(n\) をすべて求めよ。
