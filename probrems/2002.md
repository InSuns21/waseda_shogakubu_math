# 2002年 早稲田大学 商学部 数学

## 大問1

ア〜サにはいるべき数を，マーク解答用紙の該当する数字の部分に1つだけマークせよ。

1. `students` の8文字を1列に並べるとき，同じ文字が隣り合わないような並べ方は
   \[
   \boxed{\text{アイウエ}}
   \]
   通りである。

2. \(xy\) 平面上の2つの放物線 \(y=x^2\) と \(y=-x^2+3x-2\) のどちらにも接する直線は2つある。この2つの直線の交点の \(y\) 座標は，
   \[
   \frac{\boxed{\text{オ}}}{\boxed{\text{カ}}}
   \]
   である。

3. 実数を係数とする \(x\) の3次方程式
   \[
   x^3-\sqrt{3}x^2+3x+a=0
   \]
   の3つの解の実部がすべて等しいとき，
   \[
   a=-\frac{\boxed{\text{キ}}\sqrt{\boxed{\text{ク}}}}{\boxed{\text{ケ}}}
   \]
   である。

4. \(34\) 個の正の整数からなる数列 \(a_1,a_2,\ldots,a_{34}\) に対し，項差 \(d_i\ (i=1,2,\ldots,33)\) は
   \[
   d_i=a_{i+1}-a_i>0\qquad (i=1,2,\ldots,33)
   \]
   となっている。\(d_1,d_2,\ldots,d_{33}\) の中に同じ数は高々7個しかない。このとき，\(a_{34}\) の取り得る最小の値は
   \[
   \boxed{\text{コサ}}
   \]
   である。

## 大問2

原点を \(O\) とする \(xy\) 平面上の2つの点 \(P,Q\) は，それぞれ，2つの半直線
\[
l_1:x=1,\ y\ge 0,\qquad l_2:x=-2,\ y\ge 0
\]
上の点で，\(\angle POQ=60^\circ\) となるように動く。次の問いに答えよ。

1. \(\angle AOP\) の取り得る範囲を求めよ。ただし，\(A=(1,0)\) である。
2. \(\triangle POQ\) の面積の最小値を求めよ。また，そのときの \(P\) の座標を求めよ。

## 大問3

複素数 \(z_n\ (n=1,2,3,\ldots)\) が次の式を満たしている。
\[
z_1=1,\qquad z_2=\frac12,
\]
\[
z_n z_{n+1}=\frac12\left(\frac{1+\sqrt{3}i}{2}\right)^{n-1}\qquad (n=2,3,4,\ldots)
\]
このとき，次の問いに答えよ。

1. 複素平面上に \(z_1,z_2,z_3,z_4,z_5\) を図示せよ。
2. \(z_n\) を求めよ。
3. 次の和
   \[
   \sum_{n=1}^{2002} z_n=z_1+z_2+z_3+\cdots+z_{2002}
   \]
   を計算せよ。
