# 2019年 早稲田大学 商学部 数学

## 大問1

ア〜エにあてはまる数または式を記述解答用紙の所定欄に記入せよ。

1. \(\alpha,\beta\) を実数とする。
   \[
   2\cos\alpha\sin\beta+3\sin\alpha\sin\beta+4\cos\beta
   \]
   の最小値は \(\boxed{\text{ア}}\) である。
2. \(a\) を実数とする。関数 \(f(x)=x^2+a\) に対し，方程式
   \[
   f(f(x))=x
   \]
   の実数解の個数が，ちょうど \(2\) つとなる定数 \(a\) の取り得る値の範囲は \(\boxed{\text{イ}}\) である。
3. \(x\) の整式で表された関数 \(P(x)\) は，次の条件を満たしている。  
   (i) \(P(1)=1\)  
   (ii) すべての実数 \(x\) に対し，
   \[
   x\int_1^x P(t)\,dt=(x-2)\int_1^{x+1}P(t)\,dt
   \]
   このとき \(P(x)=\boxed{\text{ウ}}\) である。
4. 次の条件を満たす整数 \(n\) を \(100\) で割った余りは \(\boxed{\text{エ}}\) である。
   \[
   n\le (5+2\sqrt{5})^{2019}<n+1
   \]

## 大問2

座標平面上において，放物線 \(y=x^2\) 上の点を \(P\)，円 \((x-3)^2+(y-1)^2=1\) 上の点を \(Q\)，直線 \(y=-x+1\) 上の点を \(R\) とする。次の設問に答えよ。

1. \(QR\) の最小値を求めよ。
2. \(PR+QR\) の最小値を求めよ。

## 大問3

各項が整数である数列 \(\{a_n\}\) が，次の条件を満たしている。

(i) \(0<a_n<a_{n+1}\quad (n=1,2,3,\ldots)\)  
(ii) すべての正の整数 \(n\) に対し，\(a_n-n^2\) は \(5\) の倍数  
(iii) \(a_{2019}=6056\)

次の設問に答えよ。

1. \(a_1\) を \(5\) で割った余りを求めよ。
2. \(a_n=2021\) となる正の整数 \(n\) を求めよ。
